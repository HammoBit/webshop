# Bit Academy Webshop

## Projectnaam
### Bit Academy Webshop

## Beschrijving
Dit is een kleine webshop gemaakt met HTML, CSS, Tailwind en JavaScript.  
Ik heb dit project gemaakt tijdens de opleiding bij Bit Academy.  
Gebruikers kunnen producten bekijken, in de winkelwagen zetten en afrekenen.  
Admin kan producten beheren en bestellingen zien.  

## Functionaliteiten

### Voor gebruikers
* Alle producten zien (6 producten bij start)  
* Producten toevoegen aan winkelwagen  
* Aantal producten in winkelwagen zien  
* Producten in winkelwagen verhogen of verlagen  
* Winkelwagen leegmaken met trash icon  
* Afrekenen met bevestiging in het Nederlands  
* Na checkout worden bestellingen opgeslagen in localStorage  

### Voor Admin
* Producten zien in tabel met ID, Naam, Prijs, Afbeelding, Edit/Remove  
* Producten toevoegen via Add Product (URL of upload)  
* Producten wijzigen via Edit Product  
* Producten verwijderen  
* Reset producten naar originele staat via products.json  
* Bestellingen bekijken in aparte tab Bestellingen  
* Login verplicht om toegang te krijgen  

## Technieken
1. HTML, CSS, Tailwind CSS  
2. JavaScript (Objects, Arrays, Functions, LocalStorage)  
3. Correct gebruik van let en const  
4. DOM manipulatie  

## Installatie / lokaal draaien
1. Clone het project of download de bestanden  
2. Open index.html in de browser  
3. Voor Admin: open admin-login.html en log in  
4. Alles werkt lokaal zonder server
