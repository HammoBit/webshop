# Projectnaam

### Bit Academy Webshop

# Beschrijving

Dit project is een kleine webshop gemaakt met HTML, CSS, Tailwind en JavaScript. Het project is gemaakt tijdens de opleiding bij Bit Academy.
Gebruikers kunnen producten bekijken, toevoegen aan de winkelwagen en afrekenen. Admin kan producten beheren en bestellingen zien.

# Functionaliteiten

### Voor gebruikers

* Alle producten zien (6 producten bij start)
* Producten toevoegen aan winkelwagen
* Aantal producten in winkelwagen zien
* Producten in winkelwagen verhogen of verlagen
* Winkelwagen legen met trash icon
* Afrekenen met bevestiging in het Nederlands
* Na checkout worden de bestellingen opgeslagen in localStorage

### Voor Admin

* Producten bekijken in tabel met ID, Naam, Prijs, Afbeelding, Edit/Remove
* Producten toevoegen via Add Product (URL of upload bestand)
* Producten wijzigen via Edit Product
* Producten verwijderen
* Reset Producten naar originele staat via products.json
* Bestellingen bekijken in aparte tab Bestellingen
* Login verplicht, voorkomt toegang zonder admin

# Technieken

1. HTML, CSS, Tailwind CSS
2. JavaScript (Objects, Arrays, Functions, LocalStorage)
3. Variabelen correct gebruik van let en const
4. DOM manipulatie
# Installatie / lokaal draaien
1. Clone het project of download de bestanden
2. Open index.html in je browser
3. Admin pagina: open admin-login.html en log in
4. Alle functionaliteiten werken lokaal zonder server
